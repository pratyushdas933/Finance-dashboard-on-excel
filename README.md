Apple Inc. (AAPL) — Financial Analysis Dashboard | Excel, Financial Modelling
Designed and built a two-dashboard Excel workbook modelling Apple's financial performance using publicly reported data (10-K/10-Q, SEC EDGAR).

Dashboard 1 (Historic): Segment revenue breakdown (iPhone, Services, Mac+iPad+Wearables), consolidated P&L, COGS analysis, and profit margin trend across FY2020–FY2024
Dashboard 2 (Projected): Forward projections through FY2027E, trailing 52-week stock price chart, and a football field valuation chart benchmarking DCF, Sum-of-the-Parts, Precedent Transactions, Consensus Research, and 52-week trading range
Applied industry-standard colour coding (blue inputs, black formulas), dynamic formula-driven outputs, and a monthly FY2024 revenue breakdown with quarterly cross-checks against actual filings
